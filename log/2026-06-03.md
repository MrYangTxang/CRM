## 四、配置代理（vite.config.js）
javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
plugins: [vue()],
server: {
proxy: {
'/api': {
target: 'http://localhost:8080',
changeOrigin: true,
}
}
}
})
重启前端服务：npm run dev

## 五、安装 axios 并测试接口
```
npm install axios
```
修改 src/App.vue 为以下内容：

```vue
<template>
  <div>
    <h1>客户列表</h1>
    <ul>
      <li v-for="c in customers" :key="c.id">
        {{ c.name }} - {{ c.phone }}
      </li>
    </ul>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import axios from 'axios'

const customers = ref([])

onMounted(async () => {
  const res = await axios.get('/api/customer/list')
  customers.value = res.data
})
</script>
```
浏览器访问 http://localhost:5173/，成功显示客户列表数据。

## 六、遇到的问题及解决
### 问题1：
npm install axios 卡住
### 解决：
按 Ctrl+C 终止，重新设置镜像源后安装成功
### 问题2：
浏览器访问前端报跨域错误
### 解决：
配置 vite.config.js 代理后重启服务
### 问题3：
Vue 页面不显示数据
### 解决：
确认后端已启动，代理路径正确，浏览器控制台无 404
